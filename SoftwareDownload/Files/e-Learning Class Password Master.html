  • 软件介绍：

    • 1.本软件可以破解”极域电子教室学生端“的密码，电脑课爽玩不是梦；

    2.本软件无需安装，双击直接运行；

    3.至于如何脱离控屏请往下阅读。

  • 使用方法：

    • 方法1：

    1.运行程序，点击"显示密码"。

    2.点击任务栏中极域电子教室图标，弹出菜单后，点击"设置"。

    3.然后会弹出"设置"页面,点击"卸载"。(请注意：如果确认该电脑有保护卡或还原软件再进行该操作，否则后果，嘿嘿，你懂的。电脑如果没有保护卡或还原软件的，请阅读"方法2")

    方法2：

    1.运行程序，点击"显示密码"。

    2.点击任务栏中极域电子教室图标，弹出菜单后，点击"设置"。

    3.然后会弹出"设置"页面,勾掉"进程保护"，保存设置。

    4.同时按住键盘上"Windows"键和"R"键，打开"运行"对话框。

    5.输入： cmd ，回车。

    6.在"命令提示符"窗口中输入 taskkill /im studentmain.exe /f ,回车。
