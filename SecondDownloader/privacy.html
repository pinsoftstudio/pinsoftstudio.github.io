2. SecondDownloader integration module add-ons and extensions

"SecondDownloader" extensions for Edge, Chrome, and FireFox based browsers does not collect any data or information except for the following case when it collects data that is required to support SecondDownloader core functions.

When a browser starts a download that is going to be saved as a file on your local disk, and not displayed in the browser, and the file type matches file types in "Options→File Types" SecondDownloader dialog, or when you start a download by using SecondDownloader pop-up menu or SecondDownloader download panel, then "SecondDownloader" extensions for Edge, Chrome, and FireFox based browsers may collect only the data that is necessary to download this file in SecondDownloader from this particular web site. This data is necessary to start, or resume the download, or schedule this download in SecondDownloader. The data may include URL, internet address, cookies, encrypted credentials, query parameters, post data, namely all the data that the browser sends to the server when requesting the file.

This data is stored on your local computer until you delete this download from SecondDownloader list of downloads. To download a file, SecondDownloader re-sends the original download query made by your browser. SecondDownloader sends the data only to the server where the browser sent it to, SecondDownloader DOES NOT send this data to our servers, or any 3rd party servers.
